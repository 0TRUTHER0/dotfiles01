## What is on hyprland.conf?
### input section:
#### - Make CapsLock Key useful !
`kb_options = caps:swapescape`
#### - Type faster with:
`repeat_rate = 30
repeat_delay = 300`
### general + master
#### - Master layout ( for those who like dwm way of splitting windows )
`layout = master`
